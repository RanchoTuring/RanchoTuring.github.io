标志位

15 11 10 9 8 7 6 4 2 0
OF DF IF TF SF ZF AF PF CF
共9个 1 0
CF 进位标志（carry） 运算时在最高位(符号位也算)有借位/进位 无借位/进位
PF 奇偶（parity） 操作结果中1有偶数个（低8位） 奇数个
AF 辅助进位（auxiliary carry） 字节运算时，低半字节向高半字节进位或借位 无进位/借位
ZF 零标志 操作结果为0 不为0
SF 符号（sign） 运算结果为负（无符号数运算时SF会变，但无意义） 正
TF 单步/跟踪（trace） 单步执行（在调试时可直接控制CPU状态） 连续执行
IF 中断允许（interrupt enable） 允许CPU响应外部可屏蔽中断请求（每执行一条指令之后，自动产生一次内部中断） 不允许
DF 方向 （direction） 字符串操作时从高地址向低地址方向进行 低->高
OF 溢出 （overflow） 有符号数运算时，结果超过了寄存器容纳范围

内存地址的计算

起始地址 = 段地址 * 10H

物理地址 = 起始地址 + 偏移地址

MOV 的使用

  • mov 寄存器 ，(立即数/寄存器/内存单元/段寄存器）

  • mov 内存单元，寄存器/立即数

  • mov 段寄存器，寄存器

不允许 mov 内存单元，内存单元

XCHG 的使用

  • 必须有一个操作数在通用寄存器中

  • 不能在内存单元之间直接交换数据；不能与段寄存器交换数据

  • 数据类型必须一致

地址传送指令

  • LEA 16位寄存器，（存储器操作数/偏移地址） 将存储器操作数的地址偏移量传送到目的操作数

    设（BX）=1000H,(DI)=0100H

    LEA BX,[BX+DI+0010H] 执行后，EA=1110H

    故（bx）=1110H

  • LDS (SI/BX)，内存变量 将内存变量指向的连续4个字节内容送到DS和目的操作数中

  • LES DI,内存变量 ES和目的操作数

标志传送指令

  • LAHF flag的低八位内容->ah

  • SAHF ah->flag的低八位内容

  • PUSHF POPF 标志进出栈

输入输出指令

  • IN （al/ax），(立即数/DX) 从指定端口地址中读入一个byte/word/dword，并传送到al/ax中
  • out (立即数/dx)，(al/ax) 将al/ax中的数据输出到指定端口中

ADD 的使用

  • add 寄存器，（数据/寄存器/内存单元）
  • add 内存单元，寄存器

堆栈

  • SS:SP 始终指向栈顶

    栈空时，SS:SP指向栈空间最后地址单元的下一单元

    e.g. 将10000H~1000FH这段空间当作栈，

    SS = 1000H SP = 10H

  • POP和PUSH操作2 byte的数据

  • 由高字节向低字节压栈（随着数据增多，SP逐渐减小）

    新数据入栈之后，SP = SP - 2

寻址

操作数
立即寻址 立即数 mov ax,1234h
寄存器寻址 寄存器 mov ax,bx

存储器寻址

有效地址EA=基址（bx/bp）+变址(si/di)+位移量(8/16位带符号数)
(用BP则默认操作数在SS,否则在DS)
直接寻址 偏移地址 mov ax ,[1234H]
寄存器间接寻址 操作数在存储器中，但其偏移量在寄存器里 mov ax,[bx]
寄存器相对寻址 mov ax,[bx+1000h]
基址变址寻址 mov ax,[bx+si]
相对基址变址寻址 mov ax,[bx+si+2000h]

指令

加法指令

  • ADD/ADC 通用寄存器，（通用寄存器/内存操作数/立即数）
  • ADD/ADC 内存操作数，（通用寄存器/立即数）

ADC 的结果需要在目的操作数+源操作数的基础上再加上执行前的CF

  • inc 加一指令（CF不受影响） 可以为任何8/16/32位通用寄存器或存储器操作数

减法指令

  • sub/sbb 通用寄存器，通用寄存器/内存操作数/立即数
  • sub/sbb 内存操作数，（通用寄存器/立即数）

sbb 的结果需要在目的操作数-源操作数的基础上再减去执行前的CF

  • cmp 目的操作数-源操作数（不保留结果，只保留对标志寄存器的影响）

  • dec 减一指令（cf不受影响）可以为任何8/16/32位通用寄存器或存储器操作数

  • Neg 取补

乘法（只影响C、O，其余四种未定义）

  • 无符号 MUL

    • 字节相乘 ax<– al * 8位操作数
    • 字相乘 (dx,ax)<– ax * 16位操作数

  • 带符号 IMUL

除法（6种标志位都没有定义）

  • div

    • 字÷字节 ax÷8位操作数=al余ah
    • 双字÷字 (dx,ax)÷ 16位操作数=ax余dx

  • 带符号 idiv

类型转换指令

  1. CBW (convert byte to word)

    AL中的有符号数扩展到AX

    • AL最高位为1，AH<–0FFH
    • AL最高位为0，AH<–00H

  2. CWD(word to double word)

    AX (dx,ax)

    • ax 1 dx<–0ffffH

    • ax 0 dx<–0000H

逻辑操作

对标志寄存器的影响
and、test、or、xor of=0 cf=0 其余四种没有定义
not 无影响

test 也是目的与，但不保留结果，只保留对寄存器的影响

移位运算

  • 助记符 （通用寄存器/存储器操作数），（立即数n/寄存器CL） ;移位次数为n或CL

    8086中，n只能为1，移位次数大于1时一定要用CL
逻辑左移 SHL 最高位->CF 0->最低位 其余左移
逻辑右移 SHR 0->最高位 最低位->CF 其余右移
算术左移 SAL 最高位->CF 0->最低位 其余左移
算术右移 SAR 新最高位=原最高位 最低位->CF 其余右移

  • 移动次数大于0时，影响CSZP（A未定义），大于1时OF不受影响

  • 次数=1时，O=C xor M ;M为移位后操作数的最高位
循环左移 ROL 最高位->CF和最低位 其余左移
循环右移 ROR 最低位->CF和最高位 其余右移
带进位循环左移 RCL 最高位->CF CF->最低位 其余左移
带进位循环右移 RCR 最低位->CF CF->新最高位 其余右移
  • 移动次数大于0时，影响C（SZPA未定义），大于1时OF不受影响
  • 次数=1时，O=C xor M ;M为移位后操作数的最高位

字符串操作

MoVS lods stos cmps (都可用B、w、d后缀)

从ds:si 移动到es:di

  • cmps repe/repz 找不相等的
  • scas repne/repnz 找相等的
  • ins 目标串，dx 用来修改

    控制转移类指令

    转移指令

  • jmp 16位（段内）/32位（段间）的除立即数之外的任何有效方式

条件转移指令

  • 根据flag或CX的状态，决定是否转移
  • 对于8086，只能是短程转移 -128~127
  • 对于无符号数
Above below equal not
大 小 等 非
1 0
cf < >=
zf = !=

  • 有符号数

    | greater | less | equal | not |
    | ——- | —- | —– | —- |
    | | | | |

循环指令

  • LOOP 标号

    每循环一次，CX的值减一；当CX=0时，停止循环

  • LOOPZ/LOOPE 标号

    当CX!=0且zf=1时循环

  • LOOPNZ/LOOPNE 标号

    当CX！=0且zf=0时循环

调用返回指令

  • 段内 call (近程过程标号/16位通用寄存器/16位存储器操作数)
  • 段间 call(远程过程标号/32位存储器操作数)
  • REt 返回过程
  • int 中断
